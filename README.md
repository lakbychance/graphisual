# Graphisual

An interactive web app to draw graphs and visualize algorithms.

## Features

- Create and edit graph nodes and edges
- Visualize algorithms with auto or step-through modes
- Generate graphs from templates or custom parameters
- Pan, zoom, undo/redo support

## Algorithms

- BFS & DFS traversal
- Dijkstra's shortest path
- Prim's minimum spanning tree
- Cycle detection
- Floyd-Warshall all-pairs shortest paths

## Tech Stack

React 19 · TypeScript · Vite · Zustand · Tailwind CSS · Radix UI

## Getting Started

```bash
npm install
npm run dev
```
